Whenever the game is over(tie or someone wins), the user must click back to the options screen to start a new game.
The tournament mode keeps track of each players wins, loses and ties;
The tournament stats will only be visible when the tournament mode is selected
The tournament mode stats will be kept as long as the user don't click start game. In order to keep the tournament stats,
the user must click Tournament everytime and to remove all stats simply click start game.
